# Проект UI автотестов с Allure

В проекте реализованы UI-автотесты на Python с использованием `pytest`,
`selenium` и `allure`.

---

## Запуск тестов для формирования Allure-отчета

Для запуска тестов и формирования данных для Allure-отчета выполните
команду:

```bash
pytest -v --alluredir=allure-results
```

После выполнения команды будет создана папка `allure-results` с
результатами тестов.

---

## Просмотр сформированного Allure-отчета

Для просмотра отчета выполните команду:

```bash
allure serve allure-results
```

После выполнения команды отчет Allure откроется в браузере.
